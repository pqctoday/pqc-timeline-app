# Recommendation for Stateful Hash-Based Signature Schemes

## Author & Organization

**Organization:** NIST

## Scope

**Industries:** Firmware; Embedded Systems; IoT; Boot Security
**Region:** USA; Global
**Document type:** Algorithm

## How It Relates to PQC

Firmware signing, secure boot, archival

**Dependencies:** RFC 8554; RFC 8391

## Risks Addressed

**Migration urgency:** Medium
**Security levels:** L1-L5 equivalent

## PQC Key Types & Mechanisms

| Field                  | Value                                   |
| ---------------------- | --------------------------------------- |
| Algorithm family       | Hash-based (stateful)                   |
| Security levels        | L1-L5 equivalent                        |
