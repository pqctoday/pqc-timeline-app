# NIST HQC Selection — Fifth PQC Algorithm

**Source**: https://www.nist.gov/news-events/news/2025/03/nist-selects-hqc-fifth-algorithm-post-quantum-encryption
**Announcement date**: March 11, 2025

## Summary

HQC (Hamming Quasi-Cyclic) selected as backup KEM algorithm alongside ML-KEM (FIPS 203).
Mathematical basis: error-correcting codes (different from ML-KEM's structured lattices).
